### Hi there 👋

I'm Stan, a software engineer interested in all parts of the stack, from low-level system programming and infra to web development and mobile apps.

I love building and contributing to open-source projects in my free time. I also maintain a curated list of my [starred projects](https://github.com/angristan/awesome-stars).
